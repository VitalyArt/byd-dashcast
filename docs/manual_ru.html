1. Обзор

DashCast — Android-приложение, которое позволяет вывести любое приложение с экрана мультимедиа на цифровую панель приборов вашего BYD. Навигация, музыка, видео — всё, что работает на центральном экране, можно перенаправить на дисплей перед водителем.

  • ✅ Совместимо с BYD Seal EU (DiLink 3.0, прошивка Di3.0 / 6125F)
  • ✅ Не требует модификации системы
  • ✅ Локальный ADB через TCP (localhost) — после настройки ПК не нужен
  • ✅ Автоматически определяет внешнее закрытие приложения
💡 Предварительное условие: включите отладку ADB TCP в Настройки → Для разработчиков → Беспроводная отладка (или «ADB по сети»). Это нужно сделать один раз. При первом запуске DashCast появится окно «Разрешить отладку USB?» — нажмите Всегда разрешать с этого компьютера.

4. Проекция приложения на панель приборов

1
Нажмите «Включить проекцию» (синяя кнопка). Статус изменится на «Запуск кластера…». Устанавливается локальное ADB-подключение, и кластер переходит в режим проекции.
2
Нажмите нужное приложение в списке. DashCast переносит приложение на дисплей кластера. Статус изменится на «Панель приборов: [Имя приложения]».

Появится панель управления внизу экрана, с которой можно управлять выведенным приложением с основного экрана.

5. Во время проекции — панель управления

Когда приложение активно на кластере, внизу главного экрана появляется тёмная панель с тремя функциями дистанционного управления:

5.1 Зеркало (📺 Зеркало)

Нажмите 📺 Зеркало в строке состояния, чтобы увидеть живую копию содержимого кластера внутри DashCast. С копией можно взаимодействовать касаниями — события передаются на дисплей кластера.

Зеркало использует SurfaceControl для захвата дисплея. Если оно недоступно, включается запасной режим со скриншотом каждые 2 секунды.

5.2 Режим разделения (⬛⬛ Разделить)

Нажмите ⬛⬛ Разделить, чтобы разделить кластер между двумя приложениями:

  • Полный экран — одно приложение занимает весь кластер
  • ⬜⬛ Слева (50%) — основное приложение слева, второе справа
  • ⬛⬜ Справа (50%) — основное приложение справа

5.3 Скрыть панель

Нажмите Скрыть ▼, чтобы свернуть панель управления и видеть весь список приложений.

Настройки

7.1 Тип кластера

Выберите размер экрана вашей панели приборов. Для BYD Seal EU выберите 12,3 дюйма (cmd=30).

7.2 Поля дисплея (Overscan)

Настройте поля, чтобы вписать содержимое в видимую область экрана кластера. У кластеров с изогнутым стеклом физические края часто выходят за пределы полезной области.

  • Слева / Справа — горизонтальное поле (0–200 px с каждой стороны)
  • Сверху / Снизу — вертикальное поле (0–200 px сверху и снизу)

Нажмите Применить сейчас, чтобы сразу увидеть результат, если приложение уже выведено. Значения сохраняются между сеансами.

💡 Рекомендуемые значения для Seal EU: Слева/Справа = 80 px, Сверху/Снизу = 50 px.

9. FAQ и устранение неполадок

❓ Окно «Разрешить отладку USB?» не появляется

Проверьте, что отладка ADB TCP включена в настройках разработчика мультимедиа. Если пункта нет, сначала включите режим разработчика (7 раз нажмите на номер сборки в разделе «О системе»).

❓ Приложение не появляется на кластере после выбора

  • Убедитесь, что перед выбором приложения вы нажали Включить проекцию.
  • Некоторые приложения не разрешают запуск на дополнительном дисплее. Проверьте журнал ошибок.
  • Попробуйте закрыть и снова открыть DashCast, затем повторите последовательность.

❓ Содержимое обрезано или смещено на кластере

Настройте поля дисплея в ⋮ → Настройки. Увеличьте Слева/Справа при горизонтальном выходе за края, Сверху/Снизу — при вертикальном. Нажмите Применить сейчас, чтобы сразу увидеть результат.

❓ Кнопки «← Основной» и «✕» остаются видимыми после закрытия приложения

DashCast автоматически определяет завершение приложения (через мониторинг /proc). Если интерфейс завис, нажмите Остановить проекцию, чтобы принудительно сбросить состояние.

❓ После перезапуска автомобиля нужно всё настраивать заново?

Нет. Тип кластера и поля overscan сохраняются. Последнее выведенное приложение также запоминается. Может понадобиться только снова нажать Включить проекцию для ADB-подключения.
